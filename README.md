# Project:SUMIRE

Project:SUMIRE is an experimental research project focused on AI-assisted storytelling, emotional convergence, relationship dynamics, and long-form narrative stability.

This repository explores:

* Convergence Field Theory
* Emotional/relational structure design
* AI behavior differences between Claude, ChatGPT, and Gemini
* Long-form plot collapse analysis
* Low-temperature daily-life scenario construction
* Information asymmetry and emotional acceleration
* Character attraction and narrative convergence models

The goal is to investigate why current AI systems often fail in long-form narrative consistency, and to propose practical methods for more stable AI-driven creative writing.

## Core Themes

* Relationship-driven storytelling
* Narrative convergence
* Emotional tension control
* Structural stability in AI-generated fiction
* Prompt architecture for creative AI systems
